# AURUM — Visão Geral

Plataforma SaaS multi-tenant white-label com módulos (HR+, FinSphere, TrustID, VisionX, etc.), com foco em HR+ como canônico para padrões de UI, APIs, dados e integrações. Este documento resume os conceitos e referencia os demais docs.

- Multi-tenant: isolamento por `tenant_id` em todas as tabelas e claims JWT.
- White-label: branding por subdomínio (logo, palette, domínio custom).
- Hubs: HR+, FinSphere, NeoCrypto, TrustID, VisionX, etc.
- Integrações parceiras: Zetra Consig, ConsigaMais, ZetraPay (fictícios), e outros.

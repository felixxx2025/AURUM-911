# Épico — ATS (Recrutamento)

Objetivo: sincronizar vagas e candidatos do ATS para dossiê e automações de RH.

## Critérios de aceite

- Import incremental de candidatos e vagas
- Webhooks de mudanças de etapa
- Mapeamento de etapas -> estados internos
- Observabilidade de erros/latência

## Milestones

1. Sandbox (Gupy) e import inicial
2. Webhooks inbound e reprocessamento idempotente
3. UI para acompanhamento de pipeline
4. Relatórios e filtros

## Dependências

- Política de privacidade e retenção de dados de candidatos

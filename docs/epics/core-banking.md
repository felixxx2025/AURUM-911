# Épico — Core Banking / Infra Financeira

Objetivo: habilitar contas, cartões e transações via BaaS.

## Critérios de aceite

- Abertura/gestão de contas
- Emissão de cartões e transações
- Webhooks de transações e liquidação
- Observabilidade por operação (latência/erros)

## Milestones

1. Sandbox (Dock/FitBank/Celcoin)
2. Transações básicas e webhooks
3. Reconciliação e relatórios
4. Painel operacional

## Dependências

- Requisitos regulatórios (BACEN)
- Segurança de certificados (mTLS)
